# 🛸✨ Davinci Code (Solo vs AI) 👾🎴

ยินดีต้อนรับสู่บอร์ดเกมถอดรหัสลับ **Davinci Code** ในธีม **Cartoon Sci-Fi & Retro Toy!** 🚀 ชิปตัวเลขหนานุ่มสไตล์ของเล่นย้อนยุค พร้อมกับเหล่ามาสคอตสุดกวนที่จะมาร่วมเล่นกับคุณ!

---

## 🌟 ฟีเจอร์เด่นของเกม (Cool Features)
* 🛸 **Retro Toy Style UI:** การ์ดตัวเลขหนานุ่มเหมือนชิปพลาสติก พร้อมแอนิเมชันเด้งดึ๋งสุดน่ารัก (Elastic Bounce)
* 🧠 **AI อัจฉริยะ 3 ระดับ:** มีระดับตั้งแต่ Easy (สุ่มเดาขำๆ), Medium (เริ่มคัดตัวเลข), จนถึง Hard (คำนวณความน่าจะเป็น 100%)
* 👽 **มาสคอตขี้บ่น:** เจ้าเอเลี่ยนและคุณนักบินอวกาศลอยไปมาดุ๊กดิ๊กบนหน้าจอ และคอยกวนประสาทคุณถ้าคุณคิดนานเกิน 7 วินาที!
* ⏱️ **ระบบจับเวลาท้าทาย:** ตั้งเวลาจำกัดในแต่ละเทิร์น และมีรอบบันทึกเวลาภารกิจ
* 🃏 **Joker System:** วางการ์ดโจ๊กเกอร์ลับตรงไหนก็ได้ในมือเพื่อลวงคู่ต่อสู้

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* **Core:** HTML5 & Vanilla JavaScript (เขียนสด ไร้เฟรมเวิร์ก โหลดไวมาก!)
* **Styling:** CSS3 (การ์ตูนเซลเฉด ขอบหนา เงาคมสะดุดตา)
* **Audio:** ระบบเสียงสไตล์ Retro 8-bit

---

## 🚀 วิธีเปิดเล่น (How to Play)
1. ดับเบิ้ลคลิกไฟล์ `index.html` เพื่อเล่นในเครื่องคอมพิวเตอร์ของคุณได้ทันที!
2. หรือเล่นออนไลน์ผ่านลิงก์ของ Vercel ที่คุณได้ Deploy ไว้ 💻
